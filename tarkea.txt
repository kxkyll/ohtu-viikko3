Tässä käyttökelpoisia GIT:n komentoja
-------------------------------------------------------------
git clone reponsshosoite
git pull
git add .
git commit -m "viesti mitä muutettu"
git push
git status
git branch eka  - luo eka nimisen branchin
git checkout eka - vaihtaa eka nimisen branchin aktiiviseksi
git branch  - kertoo mitä brancheja on ja näyttää mikä niistä on aktiivisena
git branch -v - kertoo mitä on commitattu mihinkin branchiin
git push origin eka - pushaa branchiin eka commitatut muutokset remote branchiin nimeltä eka
gitk --all  - näyttää branchit ja niiden tilat

mergeäminen
1) vaihda ensin se branch aktiiviseksi mihin haluat branchin mergetä (usein master)
git checkout master
2) mergeä branch (alla mergettävän branchin nimi eka) 
git merge eka 
3) voit poistaa mergetyn branchin (ei poista branchia jota ei ole mergetty)
git branch -d eka
